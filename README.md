# AI Chatbot

A conversational AI chatbot built using Streamlit and Groq API.

## Features
- Interactive chat interface
- Multiple personalities
- Chat history
- File upload support
- Download chat history
- Modern UI

## Tech Stack
- Python
- Streamlit
- Groq API
- GitHub
